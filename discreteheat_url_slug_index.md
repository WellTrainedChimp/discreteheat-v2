*Definitive page list with hierarchy levels · February 2026*

## How to read this document

Each page is classified by hierarchy level:

- **Hub** — top-level audience entry point or section landing page. Routes users into sub-sections. Not a primary AI citation target.
- **Sub-hub** — cluster landing page within a hub. Groups related leaf pages. Light content with navigation focus.
- **Leaf** — content page. The actual citation target. Contains answer capsules, proprietary data, FAQs, and structured data.

Pages are grouped by site section. Total: **~141 pages**.
